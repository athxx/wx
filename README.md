# 微信界面 - GPUI 实现

这是一个使用 Rust 的 GPUI 框架实现的仿微信界面的桌面应用程序。

## 功能特性

### 完整界面
- ✅ 窗口标题栏（最小化、最大化、关闭按钮）
- ✅ 经典版微信三栏布局
  - 左侧工具栏（用户头像、功能图标，绿色选中状态）
  - 中间会话列表（**可水平拖动调整宽度 200-400px**）
  - 右侧聊天区域（消息展示和输入，**可垂直拖动调整输入区高度 120-400px**）

### 群组支持
- ✅ 群组 2x2 头像网格显示
- ✅ 群组成员数显示（如"技术交流群 (234)"）
- ✅ 群聊消息显示发送者名称
- ✅ 群组会话列表显示发送者

### 交互功能
- ✅ 点击会话切换聊天
- ✅ 发送消息（Enter 或点击发送按钮）
- ✅ 未读消息徽章（头像右上角红色圆点）
- ✅ 消息气泡展示（自己/对方）

### 微信风格样式
- ✅ 工具栏深色背景 (#2c2c2c)，宽54px
- ✅ 会话列表白色背景，选中灰色 (#e7e7e7)
- ✅ 聊天区域浅灰背景 (#f5f5f5)
- ✅ 绿色消息气泡 (#95ec69)
- ✅ 绿色发送按钮和工具栏选中状态 (#07c160)

## 技术栈

- **GPUI** - Zed 的 UI 框架
- **gpui-component** - UI 组件库
- **Rust** - 系统编程语言
- **Chrono** - 时间处理

## 项目结构

```
src/
├── main.rs                    # 应用入口和主窗口
├── models.rs                  # 数据模型（Contact, Message, ChatSession）
└── components/
    ├── mod.rs                 # 组件模块导出
    ├── toolbar.rs             # 左侧工具栏组件
    ├── session_list.rs        # 会话列表组件
    └── chat_area.rs           # 聊天区域组件
```

## 如何运行

### 前置要求

- Rust 1.70+ 
- Windows 10+ / Linux / macOS

### 构建运行

```bash
# 构建项目
cargo build

# 运行项目
cargo run
```

### 发布构建

```bash
cargo build --release
```

## 组件说明

### ToolBar（工具栏）
- 用户头像
- 聊天、通讯录、收藏、文件、设置等功能按钮
- 支持选中状态高亮

### SessionList（会话列表）
- 顶部搜索框
- 会话列表项展示（头像、名称、最后消息、时间）
- 支持点击选中
- 悬停效果

### ChatArea（聊天区域）
- 顶部标题栏（显示当前联系人）
- 消息列表（支持滚动）
- 消息气泡（左对齐/右对齐）
- 底部工具栏（表情、文件等按钮）
- 输入框和发送按钮
